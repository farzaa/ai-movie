Go ahead and run `python for_mac.py`.

There are a few dependencies to install like openai and elevenlabs.

You'll also need to create a `.env` file like so:

```
OPENAI_API_KEY=
ELEVENLABS_API_KEY=
ELEVENLABS_VOICE_ID=
```
